# 幸福管家高保真可点击网页原型

## 打开方式

推荐本地服务方式：

```powershell
cd D:\Product\BLS\happybutler\prototype
npm run dev -- --port 4173
```

浏览器打开：

```text
http://127.0.0.1:4173/
```

## 演示路径

1. 居民端：首页 → 一键诉求 → 提交诉求 → 进度查询。
2. 后台管理：切换“后台管理” → 诉求管理 → 处理最新工单 → 派单给党员 → 标记完成。
3. 居民端：回到居民端 → 消息/进度 → 去评价。
4. 企业端：切换“企业端” → 在线申报 → 模拟上传材料 → 提交申报 → 企业中心查看记录。

## 已覆盖页面

- 居民端：首页、一键诉求、进度查询、服务事项、个人中心。
- 企业端：首页、企业服务包、政策解读、在线申报、企业中心。
- 后台管理：数据概览、诉求管理、党员管理、服务事项、企业服务包、政策管理、公告管理、用户管理、AI知识库。
